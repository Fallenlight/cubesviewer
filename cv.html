CubesViewer

Visual data analysis and explore tool

CubesViewer is a visual, web-based tool application for exploring and analyzing OLAP databases served by the open source Cubes OLAP Framework. This version contains all major feaures (except for Save/Share or Notes, which require a server backend and authentication).
